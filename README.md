# pages-wending-lab

## 项目简介

`pages-wending-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名、网站或线上服务，仅作为静态页面文件的存储和展示空间。  
所有页面均为独立开发或收集整理的 HTML 文件，可用于测试、演示、学习或参考。

## 目录说明

```
pages-wending-lab/
├── index.html          # 主页（可选，用于浏览页面列表）
├── pages/              # 存放各独立 HTML 页面的文件夹
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             # 公共资源（如 CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：每个 HTML 文件为一个独立页面，文件名按顺序或功能命名。
- `assets/`：存放多个页面共用的样式、脚本或图片资源。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可直接在浏览器中打开运行。
- 页面内容不包含任何外部依赖（除非在 `assets/` 中提供）。
- 页面仅用于展示或功能演示，不指向任何实际网站或服务。
- 归档的页面可能包含实验性功能、学习案例或临时测试页面。

## 维护说明

- 本仓库由维护者不定期更新，添加或修改页面。
- 页面内容不保证长期可用或兼容性，建议下载后自行测试。
- 欢迎提出 Issue 或 Pull Request，但请注意本仓库不涉及商业用途或推广。
- 所有页面内容仅供学习与参考，请勿用于违反相关法律法规的场景。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在浏览器中直接打开 `pages/` 目录下的 HTML 文件。
3. 或通过 GitHub Pages 等静态托管服务查看（需自行配置）。
